## Description

GP Bubble turns everyday task planning into a visual, focused experience.

Create tasks with priorities, effort estimates, due dates, and recurring schedules. On the main screen, active tasks appear as bubbles that naturally surface based on urgency and priority, helping you decide what matters next without scanning a long, noisy list.

For times when you want a clearer overview, GP Bubble also includes dedicated list views for all pending work, completed tasks, and past-due tasks. This keeps today’s workload clean while making overdue items easy to review and edit separately.

Key features:
- Bubble-based task view ordered by priority and urgency
- Pending list view for reviewing all active tasks at once
- Separate past-due view for overdue work
- One-off and recurring tasks
- Flexible effort estimates from 5 minutes to multi-hour tasks
- Daily reminders with local notifications
- Private by default: task data stays on device

GP Bubble is designed for people who want a task manager that feels lightweight, visual, and fast to review.

- Notifications are optional and only used for local reminders.
